# One Life

Making a 100LoC text-adventure game for CPP learning.
Deadline: Sunday (16/05/2026)

## Features
- Health, Attack and Defence stats
- XP System Enabling Leveling Up
- Combat System Implementing RNG
- 3+ Monsters
- A Dungeon
- One life. If user dies, starts from beginning
- Win Condition: Defeat the final boss in the Dungeon
